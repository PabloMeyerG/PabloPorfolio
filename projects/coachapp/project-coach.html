Coach

UI · App fitness + nutrición · 2024

El reto

El reto era diseñar una app que integrara en un solo sitio rutinas de ejercicio y control calórico, sin que la interfaz se sintiera cargada. Además, tenía que ser sencilla de usar a diario y permitir entender el progreso de un vistazo.

El objetivo

Crear una aplicación que ayudara a los usuarios a entrenar de forma eficiente y a monitorizar calorías quemadas y consumidas desde un mismo lugar. La idea era ofrecer una visión integral (actividad + nutrición) con una experiencia clara y motivadora, incorporando también funciones sociales para compartir progreso.

El proceso

1. Investigación

Empecé con desk research y análisis de apps del sector para detectar patrones, puntos de fricción y oportunidades. Revisé referencias como Fitness+, Nike NTC, Centr y MyFitnessPal para entender qué valora el usuario y qué se puede simplificar.

2. Definición del usuario

A partir de la investigación definí perfiles de usuario para priorizar necesidades reales: claridad visual, rapidez de uso y seguimiento del progreso diario.

3. Estructura y flujo

Organicé la arquitectura de la información y los flujos principales: onboarding, entrenamiento diario, registro de comidas, progreso y apartado social.

4. Diseño y prototipado

Pasé de wireframes a un sistema visual en modo oscuro con acentos de color y componentes reutilizables. Finalmente, lo llevé a un prototipo de alta fidelidad con las pantallas clave.

5. Testing

Probé el prototipo con usuarios a través de tareas concretas (registrarse, iniciar entrenamiento, añadir comida). El feedback confirmó la base del diseño y marcó una mejora clara: simplificar el flujo de añadir alimentos.

El resultado

Un prototipo funcional de Coach con una interfaz clara y un sistema visual consistente, diseñado para que el usuario entienda su estado diario en segundos (actividad + nutrición + progreso).

Este proyecto me ayudó a reforzar una forma de trabajar muy práctica: investigar, ordenar la estructura, diseñar un sistema y validar con usuarios para iterar con criterio.
