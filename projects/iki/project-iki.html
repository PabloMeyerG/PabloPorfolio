IKI

Branding · Identidad visual · 2024

El reto

Crear una marca masculina minimalista con una identidad clara y reconocible en un mercado saturado, evitando soluciones genéricas y construyendo un sistema sobrio, coherente pensado para perdurar en el tiempo.

El objetivo

El objetivo fue crear un sistema visual sólido, capaz de transmitir calma, equilibrio y calidad, y de adaptarse de forma coherente a todos los puntos de contacto de la marca, tanto físicos como digitales.

El proceso

El proyecto partió de la definición del posicionamiento y los valores de la marca: simplicidad, funcionalidad y elegancia atemporal. A partir de esta base, trabajé la identidad gráfica, el sistema tipográfico y la dirección visual, buscando siempre un equilibrio entre expresión y contención.

El diseño del logotipo se desarrolló desde una construcción racional, priorizando proporción, legibilidad y personalidad, para que funcionara tanto como elemento protagonista como dentro de un sistema más amplio.

El resultado

El resultado es una identidad visual coherente y preparada para crecer, capaz de adaptarse a distintos soportes y contextos sin perder personalidad. Un sistema que prioriza la claridad, la consistencia y la experiencia de marca, tanto en comunicación como en producto.

Este proyecto me permitió profundizar en la creación de sistemas visuales pensados a largo plazo, donde cada elemento tiene un propósito y contribuye a una identidad clara y consistente.
